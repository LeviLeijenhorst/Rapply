CoachScribe – Complete Product Description

WHAT COACHSCRIBE IS

CoachScribe is a web-based platform built for coaches to record
sessions, automatically generate structured reports, and manage all
session information securely in one place.

Core promise: Better coaching begins with full attention.

It removes the need for manual note-taking during sessions so coaches
can stay fully present while maintaining professional documentation.

------------------------------------------------------------------------

TARGET AUDIENCE

Built for a broad range of coaches including: - Loopbaan coaches - Life
coaches - Business coaches - Budget coaches - Leadership coaches -
Studenten coaches - Mental health coaches - Jongeren coaches - Other
coaching professionals

Suitable for: - Independent coaches - Coaches working within
organizations

------------------------------------------------------------------------

CORE WORKFLOW

1.  Create a new session
2.  Record audio directly in the app
3.  Automatically generate a transcript
4.  Generate a structured report using a template
5.  Edit the summary
6.  Add notes
7.  Ask AI questions about the session
8.  Share a branded report with the client

------------------------------------------------------------------------

WEB APP FEATURES

Sessions Management - View all sessions - Filter (All sessions / Loose
sessions) - Search sessions - Assign sessions to coachees - View
transcript status - Edit sessions - Create new sessions - Usage counter
(minutes)

Audio Recording - One-click recording - Waveform visualization - Play /
pause - Skip 10 seconds forward/backward - Timestamp display

Automatic Transcription - Transcript generated automatically -
Generation status indicator - Option to cancel generation

AI-Generated Summaries - Structured reports based on templates - Default
structure includes: - Summary - Bullet points - Action points - Rich
text editor with: - H1 / H3 - Bold - Lists - Indentation

Template System Available templates: - Standaard samenvatting - SOAP -
Intake - Eindverslag - Voor de coachee

AI Chat (Snelle vragen) - Ask contextual AI questions - Answers
generated based on transcript - Clear indication that responses are
AI-generated

Notes - Separate section for manual notes

Coachee Management - Add coachees - Assign sessions - Organize sessions
per client

Branding (Mijn brand) - Brand name - Website - Logo upload - Brand color
selection

Shared reports reflect the coach’s branding.
